### case 1
#### 대진조건
1. 각 대진숫자 / 2의 올림수가 같으면 해당 라운드에 대진한다.
    round 1 : 3 / 5
    round 2 : (1.5) -> 2 / (2.5) -> 3
    round 3 : (2) -> 1 / (1.5) -> 2
   따라서, round 3에 match !

    - 두 숫자 중 큰 값이 짝수면 match
    - 위 조건이 성립되지 않는다면, 다음 라운드
