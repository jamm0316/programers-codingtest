## 풀이계획(*이게 제일 중요)
### 전제
1. **최대한 순간이동으로 움직이고, 점프는 최소화 할 수록 효율적임**
    - 여기서, 주의할 점은 **순간이동**은 온 만큼을 더 가는 것이 아니라, **해당 위치** 로 순간이동 하는 것임
      - ex) 점프 3번, 순간이동 1번이면 => 6번칸에 위치 (3(점프) + 6(순간이동) = 9번 아님)
3. **역순으로 구하기**
    - 짝수일 경우 : 순간이동하기
    - 홀수일 경우 : 1칸 앞으로 가기
    ex) N = 10 일때, 0, 점프(1), 순간이동(2), 순간이동(4), 점프(5), 순간이동(10)
